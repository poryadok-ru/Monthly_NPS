# Monthly NPS — Расчёт NPS по отзывам WB и Ozon

Набор Python-скриптов для автоматической обработки отзывов с маркетплейсов Wildberries и Ozon, расчёта NPS (Net Promoter Score) и формирования итоговых Excel-отчётов.

---

## Что делают скрипты

1. Читают все `.xlsx` и `.csv` файлы с отзывами из папки `Исходники WB и Ozon`
2. Объединяют с карточкой товара (`Карточка товара.xlsx`) — обогащают данные продуктом, менеджером, поставщиком
3. Классифицируют оценки: оценка 5 → Хорошая, 4 → Нейтральная, 1–3 → Плохая
4. Рассчитывают NPS по каждому товару в разбивке по годам/месяцам: `NPS = (Хорошие − Плохие) / Всего`
5. Формируют сводную таблицу (пивот) и лист с комментариями
6. Оформляют итоговый Excel: цветовая разметка, группировка строк, выделение последнего месяца жёлтым

---

## Скрипты

| Файл | Период | Фильтр NPS > 60% |
|------|--------|-----------------|
| `nps full_final.py` | Весь период (по годам) | Нет |
| `nps full_60_final.py` | Весь период (по годам) | Да |
| `nps 3mounth_final.py` | Последние 3 месяца | Нет |
| `nps 3mounth_60_final.py` | Последние 3 месяца | Да |
| `nps 9mounth 2 (final).py` | Последние 9 месяцев | Нет |

Скрипты с `_60_` в названии дополнительно удаляют из отчёта товары, у которых NPS выше 60% (т.е. всё хорошо — не требуют внимания).

## Структура папки с исходниками

```
Исходники WB и Ozon/
├── Карточка товара.xlsx       # Справочник товаров (обязателен)
├── отзывы_wb_январь.xlsx      # Любые xlsx/csv с отзывами
├── отзывы_ozon_февраль.csv
└── ...
```

Файлы с отзывами должны содержать столбцы:

| Столбец | Описание |
|---------|----------|
| `COMMENT` | Текст отзыва |
| `CREATED_AT` | Дата отзыва |
| `PRODUCT.CODE` | Код товара |
| `RATING` | Оценка (1–5) |
| `SOURCE` | Источник (WB / Ozon) |
| `IS_ABOUT_DELIVERY` | Отзыв о доставке |

## Запуск

```bash
python "nps full_60_final.py"
```

Итоговый файл появится в папке `Исходники WB и Ozon/`.

---

## Результат

Скрипт создаёт `.xlsx`-файл с двумя типами листов:

- **`общее`** — все отзывы с датами, оценками, классификацией
- **`по кодам`** — сводная NPS-таблица по товарам в разбивке по годам, с итоговой строкой и цветовым оформлением
